Save Changes to Vercel Permanently

Vercel resets its database on every cold start. To make your admin changes permanent:

  1. Make all your changes in the admin panel (upload images, update text, etc.)
  2. Click Export DB in the sidebar to download the live database
  3. Replace the db.sqlite3 file in your project folder with the downloaded file
  4. Commit and push to GitHub — Vercel will redeploy automatically
