# 🌐 Aravind Portfolio Website

A personal portfolio website built to showcase my projects, skills, certifications, and experience.

## 🚀 Features

- 3D Responsive design
- About Me section
- Skills showcase
- Project portfolio
- Certifications
- Contact section

## 🛠️ Tech Stack

- Flask
- HTML
- CSS
- JavaScript
- Bootstrap

## ▶️ Run Locally

1. Clone the repository

```bash
git clone https://github.com/aravindthiru29/Aravind-portfolio-website.git
```

2. Navigate to the project folder

```bash
cd Aravind-portfolio-website
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
